# Amazon Now

A full-stack quick-commerce web app built with React, Node.js, Express, and SQLite.

## Features
- Natural language search
- JWT authentication
- Real-time cart
- Order history & reorder
- Profile management
- Help center

## Tech Stack
- **Frontend:** React, Vite, Axios, React Router
- **Backend:** Node.js, Express, SQLite (node:sqlite)
- **Auth:** JWT + bcrypt

## Setup
